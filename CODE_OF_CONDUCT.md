# Code of Conduct

## Our Standard

This project expects respectful, constructive collaboration.

Examples of behavior that help:

- giving actionable technical feedback
- assuming good intent while still being precise
- documenting evidence when reporting bugs
- disagreeing on implementation without making it personal

Examples of unacceptable behavior:

- harassment or intimidation
- insulting or dismissive language
- bad-faith review behavior
- sharing private information without permission

## Enforcement

Project maintainers may remove comments, reject contributions, or block participation when behavior harms the project or its contributors.

## Reporting

If you need to report a conduct issue, contact the maintainer privately instead of starting a public thread.
